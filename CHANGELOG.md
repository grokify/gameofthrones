## [v0.5.0] - 2026-04-19

### Highlights

- Unified Cobra CLI (`got`) replacing 5 separate command binaries
- Comprehensive unit tests for characters, organizations, and demo data generation

### Breaking

- CLI commands moved from separate binaries to unified `got` command. Install with: `go install github.com/grokify/gameofthrones/cmd/got@latest` ([`527d82e`](https://github.com/grokify/gameofthrones/commit/527d82e))

### Added

- `got` CLI with Cobra for unified command interface ([`527d82e`](https://github.com/grokify/gameofthrones/commit/527d82e))
- `got show characters` command with `--query` and `--format` flags ([`527d82e`](https://github.com/grokify/gameofthrones/commit/527d82e))
- `got show organizations` command with `--query` and `--format` flags ([`527d82e`](https://github.com/grokify/gameofthrones/commit/527d82e))
- `got salesforce` subcommands: create-accounts, create-contacts, create-cases, delete-accounts, delete-contacts ([`527d82e`](https://github.com/grokify/gameofthrones/commit/527d82e))
- `got pipedrive` subcommands: create-orgs, create-people, get-persons, get-org-fields ([`527d82e`](https://github.com/grokify/gameofthrones/commit/527d82e))
- `got ringcentral build` command with `--output`, `--pretty`, and `--from-file` flags ([`527d82e`](https://github.com/grokify/gameofthrones/commit/527d82e))
- GoDoc comments for all exported types and functions ([`22bb9b7`](https://github.com/grokify/gameofthrones/commit/22bb9b7))
- Named constants: `csvColumnCount`, `areaCodeStride`, `phoneNumberStart`, `defaultFallbackDomain` ([`22bb9b7`](https://github.com/grokify/gameofthrones/commit/22bb9b7))

### Changed

- Improved panic messages with descriptive context ([`22bb9b7`](https://github.com/grokify/gameofthrones/commit/22bb9b7))
- Use empty struct for set membership in `acsOrgs` map (zero memory) ([`22bb9b7`](https://github.com/grokify/gameofthrones/commit/22bb9b7))
- Salesforce screenshots moved to `docs/images/salesforce/` ([`432522d`](https://github.com/grokify/gameofthrones/commit/432522d))

### Removed

- `cmd/build_data/` directory (replaced by `got ringcentral build`) ([`527d82e`](https://github.com/grokify/gameofthrones/commit/527d82e))
- `cmd/pipedrive/` directory (replaced by `got pipedrive`) ([`527d82e`](https://github.com/grokify/gameofthrones/commit/527d82e))
- `cmd/salesforce/` directory (replaced by `got salesforce`) ([`527d82e`](https://github.com/grokify/gameofthrones/commit/527d82e))
- `cmd/read_simple/` directory (replaced by `got show characters`) ([`527d82e`](https://github.com/grokify/gameofthrones/commit/527d82e))
- `cmd/test_numbers/` directory (converted to unit tests) ([`527d82e`](https://github.com/grokify/gameofthrones/commit/527d82e))
- ~60 lines of commented-out dead code in `characters.go` ([`22bb9b7`](https://github.com/grokify/gameofthrones/commit/22bb9b7))

### Dependencies

- Add `github.com/spf13/cobra` v1.10.2 for CLI framework ([`527d82e`](https://github.com/grokify/gameofthrones/commit/527d82e))

### Documentation
