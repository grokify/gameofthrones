# v0.6.0 Release Notes

Released: 2026-07-11

## Highlights

- **Period-appropriate terminology**: New data types using Game of Thrones universe terminology (Station, Orders, Allegiance) instead of modern equivalents
- **Augmented package**: New `augmented/` subpackage for modern demo data (email, phone, risk levels) separate from canonical GoT data

## New Features

### Base Package - Canonical Data

Six new data files with period-appropriate Game of Thrones terminology:

| File | Type | Description |
|------|------|-------------|
| `station.go` | `Station` | Social rank hierarchy (Smallfolk, Knight, Lord, Great Lord, Royal) |
| `status.go` | `LifeStatus` | Character life status (Living, Deceased) based on series finale |
| `titles.go` | `Title` | Formal titles with categories (Governance, Military, Religious, Academic, Commerce) |
| `orders.go` | `Orders` | Formal organizations (Night's Watch, Kingsguard, Small Council, etc.) |
| `allegiance.go` | `Allegiance` | Feudal relationships (Liege, SwornTo) |
| `resources.go` | `Resource` | Organization assets by category (Communication, Weapons, Finance, etc.) |

### Augmented Package

New `augmented/` subpackage separates modern demo data from canonical GoT data:

```go
import "github.com/grokify/gameofthrones/augmented"

// Get characters with email and phone
chars, err := augmented.GetCharacters()

// Get resources with risk levels (1-5 scale)
resources := augmented.GetAugmentedResources("King's Landing")
critical := augmented.CriticalResources("King's Landing")

// Group characters
byOrg, _ := augmented.GetCharactersByOrganization()
byStation, _ := augmented.GetCharactersByStation()
```

**Augmented types:**

- `Character` - Embeds base character + Email, Phone fields
- `Resource` - Embeds base resource + RiskLevel (1-5)
- `Organization` - Embeds base org + AreaCode, Phone, Domain

## Breaking Changes

The following types and functions have been **removed** from the base package (use the `augmented` package instead):

| Removed | Replacement |
|---------|-------------|
| `DemoCharacters` | `augmented.Characters` |
| `GetDemoCharacters()` | `augmented.GetCharacters()` |
| `DemoOrganization` | `augmented.Organization` |
| `DemoOrganizations` | `augmented.Organizations` |
| `GetDemoOrganizations()` | `augmented.GetOrganizations()` |

Files removed:

- `characters_demo.go`
- `organizations_demo.go`

## Terminology Mapping

| Modern Term | Period Term |
|-------------|-------------|
| Departments | Orders |
| Manager Hierarchy | Liege / Sworn To |
| Rank Numbers | Station |

## Migration Guide

To migrate from deprecated demo functions:

```go
// Before
import got "github.com/grokify/gameofthrones"
demoChars, err := got.GetDemoCharacters()

// After
import "github.com/grokify/gameofthrones/augmented"
chars, err := augmented.GetCharacters()
```
